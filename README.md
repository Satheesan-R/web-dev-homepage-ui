# Lotus — Web Dev Homepage UI

A simple static homepage UI for a floral shop called Lotus. This project contains HTML, CSS and image assets used to build the landing page.

## Files

- `homepage.html` — main HTML file (open in a browser to view)
- `homepage.css` — styles for the page
- `img/` — image assets used by the page

## View locally

1. Open `homepage.html` directly in your web browser (double-click the file).

OR

2. Serve the folder with a simple HTTP server (recommended for relative paths):

```bash
# from this project directory
python -m http.server 8000
# then open http://localhost:8000/homepage.html
```

## Live Demo

- [https://web-dev-homepage-ui.vercel.app/](https://web-dev-homepage-ui.vercel.app/)

## Notes

- The footer layout is supported with the CSS in `homepage.css`.
- For small changes, edit `homepage.html` and `homepage.css` and refresh the browser.
